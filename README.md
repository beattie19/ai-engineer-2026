# AI Engineer Melbourne 2026 — Schedule Planner

A single-file, self-contained schedule viewer and personal planner for
AI Engineer Melbourne 2026 (3–4 June 2026, ACMI & Federation Square).

## Features
- Side-by-side timeline of all four rooms on a shared time axis, per day
- Tap a session for full detail; star it to add to your picks
- "My picks" view grouped by day, with time-clash detection
- Copy your picks as plain text
- Selections persist in `localStorage` on the device (nothing is uploaded)
- Works offline, no dependencies, light/dark mode

## Hosting
It's a static `index.html`, so any static host works. With GitHub Pages,
just push this repo and enable Pages (Settings → Pages → Source: main, `/`).

Schedule data sourced from data.webdirections.org. Times and rooms may
change — check the official site on the day.
